# Run

```yaml title="canary.yaml"
apiVersion: canaries.flanksource.com/v1
kind: Canary
metadata:
  name: http-check
spec:
  interval: 30
  http:
    - endpoint: http://status.savanttools.com/?code=200
      responseCodes: [200]
```

```shell-session
canary-checker run canary.yaml
```

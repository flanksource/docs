## Configuration

| Field            | Description                                                                                   | Type                                                | Required |
| ---------------- | --------------------------------------------------------------------------------------------- | --------------------------------------------------- | -------- |
| `replicas`       | Pauses the canary if set to 0.                                                                | `int`                                               |          |
| `env`            | Environment variables.                                                                        | `map[string]`<CommonLink to="authentication">`EnvVar`</CommonLink> |          |
| `schedule`       | Schedule to run checks on. Supports all cron expressions. Also supports golang duration (e.g., `@every 5m`). | `cron`                                              |          |
| `icon`           | Icon to use for the check.                                                                    | `string`                                            |          |
| `severity`       | Severity of the check.                                                                        | `string`                                            |          |
| `owner`          | Contact person responsible for the canary.                                                    | `string`                                            |          |
| `resultMode`     | Result mode of the check.                                                                     | `string`                                            |          |
| `agentSelector`  | Specifies agents that should run this canary. Supports prefix/suffix (e.g., `eu-west-*`) and negations (e.g., `!team-b`). | `[]string`                                          |          |

ntfy connections enable sending push notifications via the ntfy pub/sub notification service.

## Used By

- [Notifications](/docs/guide/notifications) - Send push notifications
- [Notification Action](/docs/guide/playbooks/actions/notification) - Send notifications from playbooks

## Examples

<details summary="Public ntfy.sh">
<div>

```yaml title="ntfy-public.yaml"
apiVersion: mission-control.flanksource.com/v1
kind: Connection
metadata:
  name: ntfy-alerts
spec:
  ntfy:
    host: https://ntfy.sh
    topic: my-alerts-topic
```

</div>
</details>

<details summary="Self-Hosted with Authentication">
<div>

```yaml title="ntfy-self-hosted.yaml"
apiVersion: mission-control.flanksource.com/v1
kind: Connection
metadata:
  name: ntfy-private
spec:
  ntfy:
    host: https://ntfy.internal.example.com
    topic: infrastructure-alerts
    username:
      valueFrom:
        secretKeyRef:
          name: ntfy-credentials
          key: username
    password:
      valueFrom:
        secretKeyRef:
          name: ntfy-credentials
          key: password
```

</div>
</details>

Pushover connections enable sending push notifications to iOS and Android devices via the Pushover service.

## Used By

- [Notifications](/docs/guide/notifications) - Send push notifications
- [Notification Action](/docs/guide/playbooks/actions/notification) - Send notifications from playbooks

## Example

```yaml title="pushover-connection.yaml"
apiVersion: mission-control.flanksource.com/v1
kind: Connection
metadata:
  name: pushover-alerts
spec:
  pushover:
    token:
      valueFrom:
        secretKeyRef:
          name: pushover-credentials
          key: API_TOKEN
    user:
      valueFrom:
        secretKeyRef:
          name: pushover-credentials
          key: USER_KEY
```

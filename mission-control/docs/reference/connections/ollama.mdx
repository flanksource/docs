Ollama connections enable integration with locally hosted Ollama instances for AI-powered playbook actions without external API dependencies.

## Used By

- [AI Action](/docs/guide/playbooks/actions/ai) - Use self-hosted AI for log analysis and automation

<Fields connection="ollama"/>

## Example

```yaml title="ollama-connection.yaml"
apiVersion: mission-control.flanksource.com/v1
kind: Connection
metadata:
  name: ollama-local
spec:
  ollama:
    model: llama3.1:8b
    url:
      value: http://ollama.ollama-system.svc:11434
```

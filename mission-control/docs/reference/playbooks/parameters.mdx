## Defaulting

Parameter values can be defaulted from the selected resource

```yaml title="default parameters.yaml"
#...
kind: Playbook
spec:
  parameters:
    // Use the config items type and name in the parameter
    // highlight-next-line
    - default: 'chore: update $(.config.type)/$(.config.name)'
```

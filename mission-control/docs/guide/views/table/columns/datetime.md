The `datetime` column type displays timestamps with human-readable formatting.

## Properties

| Property | Type   | Description        |
| -------- | ------ | ------------------ |
| `type`   | string | Must be `datetime` |

See [Common Properties](/guide/views/table/columns) for the full list of fields available on all column types.

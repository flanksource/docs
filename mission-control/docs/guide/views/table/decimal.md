The `decimal` column type displays precise floating-point numbers with configurable decimal places.

## Properties

| Property    | Type    | Description                   |
| ----------- | ------- | ----------------------------- |
| `precision` | integer | Number of decimal places      |
| `unit`      | string  | Display unit (e.g., "$", "%") |

# Bark

Bark is a push notification service for iOS devices.

:::info URL Format
bark://**`devicekey`**@**`host`**
:::

## Example

```uri
bark://DEVICEKEY@api.day.app/?title=Alert&body=Check+failed&group=monitoring
```

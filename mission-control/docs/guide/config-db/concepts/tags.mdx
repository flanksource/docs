Catalog items can have tags attached to them.

For [AWS](/docs/guide/config-db/scrapers/aws), these can be `zone-id`, `region` and `account`.
[Kubernetes](/docs/guide/config-db/scrapers/kubernetes) resources have `cluster` and `namespace` tags associated with them.

The tags for any catalog item are visible on its page in the sidebar.

<Screenshot img="/img/tag-sidebar.png" />

## Usage

### Filters

Tags help filter catalog items in UI (inclusion and exclusion)

<Screenshot img="/img/tag-filters.png" />


### RBAC

[RBAC](/docs/reference/rbac) rules can use tags to fine-grain access to limited resources.

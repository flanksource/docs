# Telegram


Telegram messages are sent with `parseMode=MarkdownV2`


:::info URL Format
    telegram://__`token`__@telegram?chats=__`channel-1`__[,__`chat-id-1`__,...][&notification=no[&preview=false][&parseMode=html]
:::

:::


## Getting a token for Telegram

1. Talk to [the botfather](https://core.telegram.org/bots#6-botfather).
2. See [getting a chat id](https://github.com/GabrielRF/telegram-id).
